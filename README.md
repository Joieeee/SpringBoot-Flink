## 主要步骤

​		创建完SpringBoot项目且完成以上配置后，通过在dao类下定义一个接口，其中写一些自己需要使用的方法，然后通过在service下的impl包定义一个类来实现之前的接口，再通过重写接口中的方法，写入自己需要的算子转换（例如连接数据库，转换格式，更新...等等），这些都实现之后通过自动注入（`Autowired`）注释将该类注入到application类中，通过调用类的方法来完成SpringBoot与Flink代码的简单集成。
